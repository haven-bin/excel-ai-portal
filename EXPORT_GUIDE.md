# 数据导出功能使用指南

## 概述

在分析完数据后，可以将分析结果导出为 Excel、HTML 或 CSV 格式的文件。导出功能支持多种导出方式，满足不同的需求。

---

## 📥 导出功能

### 1. 导出 Excel 格式

点击结果区域的 **📊 导出 Excel** 按钮

**包含内容：**
- ✅ **样本数据 Sheet** - 上传文件的样本记录
- ✅ **统计摘要 Sheet** - 总记录数、使用模板、分析时间等
- ✅ **字段分析 Sheet** - 每个字段的详细统计信息
  - 字段名、数据类型
  - 非空数、空值数
  - 文本字段：唯一值数、平均长度、最长/最短字符
  - 数值字段：总和、平均值、最大/最小值
  - 常见值（Top Values）
- ✅ **唯一值统计 Sheet** - 按字段统计唯一值数量
- ✅ **聚合统计 Sheet** - 按字段统计合计值

**示例数据结构：**
```
Sheet: 样本数据
┌─────┬──────────┬────────┬───────────┐
│ 操作 │ 关联卡号 │ 备注   │ ...       │
├─────┼──────────┼────────┼───────────┤
│ null│ 52584...│       │ ...       │
└─────┴──────────┴────────┴───────────┘

Sheet: 统计摘要
┌──────┬──────────┐
│ 指标  │ 值       │
├──────┼──────────┤
│ 总记录数 │ 2064   │
│ 使用模板 │ generic│
│ 字段总数 │ 14    │
└──────┴──────────┘

Sheet: 字段分析
┌────────┬────────┬─────────┬─────────┬────────────┐
│ 字段名  │ 类型   │ 非空数  │ 空值数  │ 其他统计信息 │
├────────┼────────┼─────────┼─────────┼────────────┤
│ 关联卡号 │ text   │ 2064   │ 0      │ 唯一值: 386  │
│ 金额   │ text   │ 2064   │ 0      │ 唯一值: 210  │
└────────┴────────┴─────────┴─────────┴────────────┘
```

**文件命名：** `分析报告_[模板ID]_[时间戳].xlsx`

### 2. 导出 HTML 报告

点击结果区域的 **📄 导出 HTML 报告** 按钮

**包含内容：**
- ✅ 美观的 HTML 报告页面
- ✅ 统计摘要卡片展示（总记录数、字段数、使用模板、生成时间）
- ✅ 字段分析表格
- ✅ 可直接在浏览器中打开和打印

**文件命名：** `分析报告_[模板ID]_[时间戳].html`

**使用场景：**
- 👥 共享给团队（无需 Excel）
- 📨 通过邮件发送报告
- 🖨️ 直接打印为 PDF（调用浏览器打印功能）

## 🛠️ 导出函数详解

### 导出 Excel 分析结果
```typescript
import { exportAnalysisToExcel } from './exportUtils'

// 导出分析结果
exportAnalysisToExcel(
    result,                    // 后端返回的分析结果对象
    '我的分析报告'             // 文件名（可选，默认为"数据分析报告"）
)
```

### 导出 HTML 报告
```typescript
import { downloadAnalysisReport } from './exportUtils'

downloadAnalysisReport(
    result,                    // 后端返回的分析结果对象
    '我的分析报告'             // 文件名（可选)
)
```

### 导出原始数据
```typescript
import { exportRawDataToExcel } from './exportUtils'

// 将原始数据导出为 Excel
exportRawDataToExcel(
    rawData,                   // 数据数组
    ['字段1', '字段2', ...]   // 指定要导出的字段（可选）
    '原始数据'                 // 文件名
)
```

### 生成分析报告 HTML 字符串
```typescript
import { generateAnalysisReport } from './exportUtils'

const html = generateAnalysisReport(result)
// 现在 html 包含完整的 HTML 页面内容
// 可以进一步处理或保存
```

## 💡 典型使用场景

### 场景 1: 数据质量检查
1. 上传 CSV/Excel 数据
2. 选择合适的模板进行分析

4. 在 Excel 中查看各字段的缺失值、异常值等

1. 分析数据
2. 导出 Excel
3. 用 Python/R 等工具进一步处理统计数据

## 🔧 自定义导出

### 只导出特定字段

```typescript
import { exportRawDataToExcel } from './exportUtils'

// 只导出 "关联卡号", "金额", "时间" 三个字段
exportRawDataToExcel(
    result.sample_data,                    // 或完整数据
    ['关联卡号', '金额', '时间'],         // 指定字段
    '关键字段数据'                         // 文件名
)
```

### 修改导出格式

可以编辑 `src/exportUtils.ts` 中的函数来自定义导出格式：

```typescript
// 例如，只导出特定 Sheet
export function exportAnalysisToExcel(
    analysisResult: DynamicAnalysisResult,
    fileName: string = '数据分析报告'
): void {
    const workbook = XLSX.utils.book_new()
    
    // 只保留样本数据和统计摘要
    const sampleSheet = XLSX.utils.json_to_sheet([...])
    const summarySheet = XLSX.utils.json_to_sheet([...])
    
    XLSX.utils.book_append_sheet(workbook, sampleSheet, '样本数据')
    XLSX.utils.book_append_sheet(workbook, summarySheet, '统计摘要')
    
    XLSX.writeFile(workbook, `${fileName}.xlsx`)
}
```

## 💾 浏览器设置

某些浏览器可能会对下载进行限制或要求确认。建议：
- ✅ 在下载提示时选择 "保存" 或 "继续"
- ✅ 检查浏览器的下载文件夹
- ✅ 禁用可能阻止下载的浏览器扩展

## 📋 导出的 Excel 与后台数据的对应关系

| Excel Sheet | 来源数据 | 说明 |
|-------------|---------|------|
| 样本数据 | `sample_data.fields` | 第一条记录 |
| 统计摘要 | `records_count`, `analysis.analysisTime` etc. | 全局统计 |
| 字段分析 | `analysis.field_*` | 每个字段的详细分析 |
| 唯一值统计 | `analysis.unique_*` | 各字段唯一值数量 |
| 聚合统计 | `analysis.total_*` | 各字段的合计值 |

## ❓ 常见问题

**Q: 导出的文件在哪里？**
A: 下载到你的浏览器默认下载文件夹。通常是 `C:\Users\[用户名]\Downloads`

**Q: 可以修改导出的文件格式吗？**
A: 完全可以。编辑 `src/exportUtils.ts` 中的函数来自定义格式。

**Q: 支持导出 CSV 吗？**
A: 支持。使用 `exportAnalysisToCSV()` 函数。

**Q: 导出的 HTML 能否在离线情况下打开？**
A: 完全可以。HTML 文件是独立的，不需要网络连接。

**Q: 能否自动邮件发送导出文件？**
A: 需要额外集成邮件服务。当前版本只支持本地下载。
